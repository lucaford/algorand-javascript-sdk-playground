## Installation

OS X & Linux:

```sh
git clone https://github.com/lucaford/algorand-javascript-sdk-playground.git

cd algorand-javascript-sdk-playground
```

yarn

```sh
yarn install
```

npm

```sh
npm install
```

## Usage

### Start project CLI

```sh
yarn start
```

### Clean dependencies

```sh
yarn clean
```
