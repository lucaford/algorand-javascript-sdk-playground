![](./cli_image.png)

## Installation

OS X & Linux:

```sh
git clone https://github.com/lucaford/algorand-javascript-sdk-playground.git

cd algorand-javascript-sdk-playground
```

yarn

```sh
yarn install
```

npm

```sh
npm install
```

## Pre-Requisites

### Algorand Test Node w/Docker

```sh
git clone https://github.com/algorand/sandbox.git
cd sandbox
./sandbox up testnet
```

## Usage

### Start project CLI

```sh
yarn start
```

### Clean dependencies

```sh
yarn clean
```
